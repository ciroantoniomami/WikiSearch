Category Status

It is probably slightly noticeable that I am doing a lot of work on cleaning up the categories. So I thought an update on what has been done may be helpful.

Countries

Category:Countries was the first target. The goal was a more stream-lined set up from the main cat through continents to countries and country cats. With more pages in the smaller countries, new cats for them will eventually be needed to be created and linked up. All Categories applying to a country are on its category page. The only category on the page itself should be its own category but there will always be exceptions I never thought of.

  • Main work done, Maintenance as needed with new articles.

Cities

Category:Cities was next. Using countries as a basis, Cities were linked up from state/province for larger countries to country to continent. This is pretty much done. Many cities are in a cat above where they should be because there are not enough city pages in the lower levels to warrant a category for them yet. Cities are linked to both country and the level above them. (ie. Cities in France -> France and Cities in Europe). Capital cities is a special class that pretty much runs the same pattern. Cities that are large metropolitan areas (New York, London, Rome, LA) and can expect many pages related to them get their own category. Cities with multiple sports teams and/or tourist attractions seem to be getting the needed 3 pages for cat creation earliest.

  • Main work done, Maintenance as needed.

People

Category:People is the current target. The main "people by nationality" cat thread is set up almost identical to Cities and the same drawbacks are there. Athletes in major sports was next and fairly well dealt with (mainly Football/soccer, not a lot of non-football athlete pages).

I am currently focusing on individual countries to categories major groups in them (ie. American singers in American musicians in American entertainers in People from the United States) As England (more so than the entirety of the UK) and the United States have the largest number of pages to work through and many of these are only partially categories (an American actor/singer is only listed under musicians or even music..) there are a lot of categories which need to be looked at to get a good grip on this area before turning to smaller countries (page count wise) and see if there is enough for them to get categories.

  • Nationalities done
  • Athletes close to done
  • Entertainers in work.
    • Composers, Singers, Musicians, Pianists - Done (First pass).
